# Roadmap: Objectif Lycee

## Overview

Plateforme lycee centree sur une boucle simple: objectif declare, mission du jour, preuve d'impact, progression visible, abonnement si la valeur est ressentie.

## Phases

- [x] **Phase 1: Cockpit statique** - pages principales, mission du jour, progression locale.
- [x] **Phase 2: Objectif lycee** - page objectif, leviers dossier, impact Parcoursup et priorites.
- [ ] **Phase 3: Paiement Stripe** - offre 10 EUR/mois, Payment Link live, confirmation.
- [ ] **Phase 4: Onboarding utile** - objectif, specialites, echeances, premiere mission personnalisee.
- [ ] **Phase 5: Contenu et acquisition** - pages methode et contenus lies aux missions.

## Current Priority

### Phase 3: Paiement Stripe
**Goal**: valider que l'offre peut convertir avant de construire un backend.

**Success Criteria**
1. Le Payment Link Stripe live est configure.
2. Tous les CTA abonnement pointent vers le bon lien.
3. La page `merci.html` explique clairement la suite.
4. Les verifications Playwright restent vertes.
