# Plan 03-01: Landing page + integration Stripe checkout

## Goal

Livrer une page d'offre Outil Prepa avec un CTA d'achat mensuel a 10 EUR, une redirection Stripe Checkout via Payment Link, et une page de confirmation post-paiement.

## Scope

- `checkout.html`: page d'offre et CTA Stripe.
- `merci.html`: page de confirmation apres paiement.
- `scripts/checkout.js`: resolution de l'URL Payment Link, validation minimale, redirection, stockage local pour test.
- Navigation: entree "Abonnement" depuis le cockpit et la page ATS.

## Stripe contract

Cette phase reste statique et ne doit pas exposer de cle secrete Stripe.

Le checkout utilise une URL Stripe Payment Link:

- format attendu: `https://buy.stripe.com/...` ou `https://checkout.stripe.com/...`
- produit a configurer dans Stripe: abonnement mensuel 10 EUR
- success URL recommandee dans Stripe: `merci.html`
- cancel URL recommandee dans Stripe: `checkout.html`
