# Illustrations de progression du compte - bibliotheque

Ce dossier contient la serie d'images imagegen pour la progression globale du compte.

Concept retenu : un bureau compact avec une seule etagere carree en bois au-dessus. L'etagere se remplit au fil du temps pendant qu'un eleve, vu de dos, travaille dans une ambiance lo-fi calme.

## Nommage attendu

- `desk-shelf-stage-01-start.png`
- `desk-shelf-stage-02-first-books.png`
- `desk-shelf-stage-03-organized-notes.png`
- `desk-shelf-stage-04-methods-mastered.png`
- `desk-shelf-stage-05-full-shelf.png`

Passer en `.webp` avant publication si un outil de conversion est disponible.

## Regles de serie

- Garder le meme bureau, la meme etagere, le meme angle camera et la meme lumiere.
- Le personnage reste de dos pour eviter d'en faire un avatar identitaire.
- L'etagere carree est l'objet de progression principal.
- Ne pas ajouter de texte, chiffres, formules, axes, symboles mathematiques, logo ou watermark dans l'image.
- L'image ne doit pas porter seule le statut du compte : afficher le palier et le prochain objectif en HTML.

## Accessibilite

- Si l'image est decorative : `alt=""` et `aria-hidden="true"`.
- Si l'image porte le statut : alt court, par exemple `Bureau d'etude avec une etagere presque vide, palier debut`.
- Ne pas dependre uniquement de l'image ou de la couleur pour expliquer le niveau.
- Respecter `prefers-reduced-motion` si une transition de remplissage est ajoutee plus tard.
