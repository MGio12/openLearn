# Sources du prototype

Ce dossier est un prototype de rendu. Aucun PDF réel n'a encore été fourni pour cette première page d'exemple.

Quand le pipeline sera lancé sur un vrai corpus, ce fichier devra contenir :

- le nom de chaque PDF ;
- son origine ;
- le droit d'utilisation connu ;
- les chapitres ou sections réellement utilisés ;
- les points repris dans le cours final ;
- les points écartés ou jugés hors programme.

Règle : ne pas publier un cours généré depuis une source dont le droit d'utilisation est incertain.
